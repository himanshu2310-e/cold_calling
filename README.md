🚀 CodeConnect

CodeConnect is a modern, enterprise-grade Cold Calling CRM built to help sales teams efficiently manage leads, track customer interactions, organize follow-ups, upload call recordings, and monitor team performance through a premium analytics dashboard.

Designed with a sleek dark-themed UI inspired by Linear, Attio, Vercel, and Stripe, CodeConnect delivers a fast, intuitive, and scalable experience for businesses handling outbound sales.

🚀 Getting Started
Clone the Repository
git clone https://github.com/your-username/codeconnect.git
Install Dependencies

Frontend

cd frontend
npm install

Backend

cd backend
npm install
Configure Environment Variables

Backend

PORT=
MONGODB_URI=
JWT_SECRET=
JWT_REFRESH_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

Frontend

NEXT_PUBLIC_API_URL=
NEXT_PUBLIC_SOCKET_URL=
Run the Application

Backend

npm run dev

Frontend

npm run dev
